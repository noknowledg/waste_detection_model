
# Smart Waste Detection using CNN

## Project Overview
This project focuses on developing a Smart Waste Detection system using a Convolutional Neural Network (CNN) to classify waste into different categories. The model helps automate waste segregation to support better recycling and environmental sustainability.

## Categories
The model classifies waste into three categories:
- Biodegradable
- Non-Biodegradable
- Hazardous

## Dataset
The dataset was collected and prepared using Roboflow. Images were preprocessed and organized into training and validation sets for model training.

## Model
- Model Type: Convolutional Neural Network (CNN)
- Built: From scratch
- Training Epochs: 35
- Final Accuracy: 83%

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Roboflow
- NumPy
- Matplotlib

## Project Workflow
1. Dataset collection and preprocessing using Roboflow  
2. Building CNN architecture from scratch  
3. Training the model for 35 epochs  
4. Evaluating model performance  
5. Predicting waste categories from images

## Results
The model achieved an accuracy of **83%** in classifying waste into biodegradable, non-biodegradable, and hazardous categories.

## Applications
- Smart waste management systems  
- Automated waste segregation  
- Environmental monitoring systems
